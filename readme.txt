== Description ==

Shortcodes Manager which allow you to enable or disable any shortcodes from third plugins and default.

* Tools -> Shortcodes Manager

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/shortcodes-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
